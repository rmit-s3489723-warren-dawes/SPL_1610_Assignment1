NAME

Extract.pl - Sample script showing implementation of the EventExtractor module.

SYNOPSIS

This script supports 3 types of usage: perl Extract.pl perl Extract.pl input.json perl Extract.pl inout.json output.json

Note:Input and output can be any file extension, but just represet JSON format.

METHODS

These are the subs which are used within this script:

CommandArgsHandler()

Checks if any arguments were supplied to the script and attempts to set an input/output file. If found, these overwrite default (emails.json and events.json).

FileOpener($inputFile)

Attempts to open the file containing all emails to process. Note:This method will die if unable to open the emails file.

PrintToConsole($string)

Print $string to console (where $string is important).
